# :construction_worker: How work ?
```bash
# Clone this repository
$ https://github.com/KaikySantos/moveit.git

# Access the project folder
$ cd moveit

# Install the dependencies
$ yarn install

# Run the application
$ yarn dev
```
Go to http://localhost:3000/ to see the result.
