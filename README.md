# :construction_worker: Como rodar ?
```bash
# Clone o Repositório:
$ https://github.com/KaikySantos/moveit.git

# Entre na pasta criada:
$ cd moveit

# Instale as dependências
$ yarn install

# Rode a aplicação
$ yarn dev
```
Entre em http://localhost:3000/ para ver o resultado.
